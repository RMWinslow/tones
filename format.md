# HTML version

Kramdown supports lazy table rendering without the typical markdown header whatzits

so 

```
|1155|665-|4433|221-|
|5544|332-|5544|332-|
|1155|665-|4433|221-|
```

yields

|1155|665-|4433|221-|
|5544|332-|5544|332-|
|1155|665-|4433|221-|

html entities can be used to add marks for octaves and eight notes

|3&#775;|`3&#775;`|
|3&#776;|`3&#776;`|
|3&#778;|`3&#775;`|
|3&#803;|`3&#803;`|
|3&#804;|`3&#804;`|
|3&#805;|`3&#805;`|
